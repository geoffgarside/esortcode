h1. eSortCode

h1. &rarr; 'esortcode'


h2. What

An interface to the "eSortCode.com":http://www.esortcode.com UK bank and credit card
validation service.

h2. Installing

<pre syntax="ruby">sudo gem install esortcode</pre>

or for the development version

<pre syntax="ruby">sudo gem install geoffgarside-esortcode -s http://gems.github.com/</pre>

h2. The basics

<pre syntax="ruby">client = Esortcode::Client.new('user', 'license_id')</pre>
